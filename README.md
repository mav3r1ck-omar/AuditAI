# AuditAI

## AI Driven SEO &amp; Content Auditor

### Concept

  A SaaS dashboard where users input a URL, and the app scrapes it to analyze SEO performance and generate AI-suggested improvements.

### The Tech Stack

    Frontend: Next.js, Recharts (for data viz), Shadcn/UI components.
    Backend: Node.js, Puppeteer (for scraping), OpenAI API.
    Tech Flex: Using Server Actions in Next.js for the analysis workflow. Implement Redis to cache results so you don't hit API limits.
    Visuals: Interactive graphs showing "Site Health," keyword density heatmaps, and a chat interface to ask the AI about the report.

### Key Features Implemented

  1. Premium SEO Dashboard
    Site Health Score: A glowing radial chart that synthesizes overall technical SEO performance.
    Keyword Density Map: A beautiful gradient-styled bar chart showing the most frequent thematic clusters on the page.
    Actionable AI Suggestions: A dedicated section for high-impact SEO improvements generated by Gemini.
  2. Interactive SEO Chatbot
    Use the Audit Chatbot on the right side of the dashboard to ask specific questions about your report (e.g., "How can I improve my description?").
    The AI has been tuned to provide plain-text, actionable advice without messy formatting.
  3. Deep Analysis Engine
    Puppeteer Integration: The engine waits for full DOM rendering to extract titles, meta tags, heading structures, and word counts.
    Content Synthesis: Gemini 2.5 Flash analyzes the text content to provide a human-like summary of site quality.
